# Geopython-101
Introducing Python and it's geospatial packages for begineer


## Installation

1. Clone this repository
```
git clone https://github.com/Rotten-Grapes-Pvt-Ltd/Geopython-101.git
```

2. Create virtualenv

```
cd Geopython-101
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
```
